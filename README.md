# momentum_clone
 momentum clone app with vanillaJS
 
### 구현 기능
  1. 시계 : clock.js
  1. 사용자 이름 : greeting.js
   - local storage에 저장
  1. 할일 리스트 : todo.js
   - local storage에 저장
  1. 임의 배경화면 : bg.js
   - local image
  1. 위치, 기온 표시 : 
   - openweathermap API
